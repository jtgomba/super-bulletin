Super Bulletin - Bug and Issue Tracker
================

**Technologies Used**
---------------------

-   TypeScript
-   React
-   MaterialUI
-   Redux
-   Recharts
-   Firebase
-   HTML5
-   CSS3

**Overview**
------------

I made this app to help to seamlessly report, monitor, and prioritize these bug as they are discovered along the way in a project. It will allow users to know the status of all bugs that have been logged with it in a clear and intuitive interface. It will also allow users to collaborate on bugs and issues through a comment system built into each ticket. 

**Functionality**
-----------------

Currently the application has been built to do the following:

-   Users can create and create projects and tickets in a kanban style interface
-   Provides charts that quickly visualise the status of projects and tickets. 
-   Fully responsive on all devices
